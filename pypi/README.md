# smbcloud-cli

Python distribution for the `smb` command line interface used to access
[smbcloud](https://smbcloud.xyz/).

## Installation

```bash
pip install smbcloud-cli
```

This package installs the native `smb` executable for the current platform.

## Usage

```bash
smb --help
```
